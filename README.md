## 🚀 About the Project
This project is a sophisticated web-based financial tool. It allows users to calculate how much cryptocurrency they can acquire based on a specific amount of fiat currency (USD, EUR, ARS). It demonstrates advanced web development skills such as asynchronous programming (Promises/Async-Await), API integration, and responsive UI design.

## ⚙️ How the Calculator Works
The Crypto Calculator connects to the **CryptoCompare API** to fetch the most accurate and up-to-date market data.

**Key features include:**
* 💹 **Real-time Pricing:** Get the exact current price of BTC, ETH, AXS, and BNB.
* 📈 **Trend Analysis:** Displays the 24H price fluctuation with visual indicators (green for up-trends, red for down-trends).
* 🔢 **Instant Conversion:** Calculate exactly how many digital assets you receive for your money.

**To use it:**
1. Enter the amount of money you wish to "give".
2. Select your local currency and the cryptocurrency you want to "buy".
3. Click on the **CALCULATE** button (enabled only when the form is valid). 
4. View the detailed breakdown including 24H highs, lows, and the final conversion result. 🏆

## 🛠️ Technologies Used
This project was built focusing on performance and clean code:
* 🟨 **JavaScript (25.8%):** Manages asynchronous API calls, form validation logic, and dynamic DOM updates for price trends.
* 🟧 **HTML (31.8%):** Defines the semantic structure and the interactive form elements.
* 🟦 **CSS (42.4%):** Implements a "Dark Mode" aesthetic with neon accents (`#20f0aa`), custom scrollbars, and a fully responsive layout.
